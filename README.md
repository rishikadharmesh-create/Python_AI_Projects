**SUMMARY**
This repository contains a collection of projects I've built to explore different areas of Python, including quantitative 
finance, machine learning, financial modelling, and data automation. Each project was created to solve a real problem 
or to deepen my understanding of a particular domain.

**PROJECTS**
1. **GARCH Sentiment Trading Strategy**
-  This project explores whether combining market volatility and social media sentiment can improve trading decisions.
-  Using intraday stock data from Polygon.io, a GARCH model is used to forecast volatility while Twitter/X sentiment provides an additional market signal.
-  These signals are combined and tested through a backtesting framework to evaluate performance over historical 
data.

2. **Investment Calculator**
-  A personal finance tool designed to estimate the future value of investments.
-  It takes into account regular contributions, annual contribution increases, compound growth, inflation, and exchange rate effects.
-  The calculator also allows comparisons between continuing contributions and stopping contributions after a chosen number of years.

3. **Stock Price Prediction**
-  This project uses technical indicators and a Random Forest model to predict potential buy and sell signals for stocks.
-  To complement the predictions, Monte Carlo simulations are used to estimate future price movements, target probabilities, and stop-loss scenarios, helping provide a broader view of potential outcomes.

4. **WhatsApp Extractor**
-  A utility built to extract call records from exported WhatsApp chat PDFs.
-  The script organises the data into a structured format, sorts it chronologically, and generates both Excel and PDF reports, making it easier to analyse communication history.
